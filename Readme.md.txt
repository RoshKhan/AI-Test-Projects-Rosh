# TaskFlow Voice AI — Multi-Agent System

A multi-agent voice AI system built with ElevenLabs 
that lets product managers query TaskFlow's product 
health data by speaking naturally.

## Architecture
- Greeter router agent
- Feedback Analyst (NPS, reviews, interviews)
- Support Analyst (Zendesk, churn risk)
- Metrics Analyst (conversion, retention, DAU/MAU)

## Knowledge Bases
Three scoped KB files — one per specialist agent.

## Built with
ElevenLabs Conversational AI
